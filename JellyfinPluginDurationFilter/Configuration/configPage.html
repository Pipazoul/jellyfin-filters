Duration Filter

Adds a Min/Max runtime filter (in minutes) to the jellyfin-web library filter panel. Changes take effect after users reload their browser tab.

Pre-fills the "Min minutes" input the first time a library is opened. 0 means no minimum. This only seeds the input; it is not applied automatically.
Pre-fills the "Max minutes" input. 0 means no maximum.
Comma-separated list of library IDs the filter should appear in. Leave empty to enable it in every video library. You can copy a library's ID from its URL in the Jellyfin dashboard, or from the address bar while browsing the library.
Displays a small "Duration: …" chip with a one-click clear button whenever a filter is applied to the library being viewed.
The plugin prefers the File Transformation plugin (clean, no file changes). If that plugin is not installed and this box is checked, the plugin instead edits jellyfin-web/index.html on disk. Uncheck it if your web root is read-only.
